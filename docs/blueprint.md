# **App Name**: Personalized Learning Nexus

## Core Features:

- User Authentication: Secure user sign-in with Google, GitHub, and email.
- Firestore Integration: Utilize Firestore for storing learning materials and user data.
- AI-Powered Quiz Generator: Leverage AI to generate personalized quizzes based on learning content. This is done via tool usage for relevant data selection.
- Personalized Study Plans: AI-driven tool to customize learning paths, adapting to each student's progress and knowledge gaps.
- Performance Metrics Dashboard: Visualize student progress, knowledge gaps, and overall learning effectiveness.
- Adaptive Feedback: Offer AI-driven, customized feedback on quizzes, enhancing student learning with individualized mentoring.
- Content Summarization: Summarize text based learning materials.

## Style Guidelines:

- Primary color: Electric Indigo (#6F00FF) to evoke innovation and a high-tech feel.
- Background color: Dark charcoal gray (#1A1A1A) to ensure contrast and readability in a dark theme.
- Accent color: Cyan (#00FFFF) for highlighting interactive elements.
- Headline font: 'Space Grotesk', a modern sans-serif font, giving a computerized feel.
- Body font: 'Inter', to complement the Space Grotesk headlines, adding balance and clarity.
- Use modern, neon-style icons that pop against the dark background.
- Modernistic design, utilizing white space and high-quality images to enhance the user experience.
- Subtle, yet impactful animations to provide feedback and guide the user through the learning process.
